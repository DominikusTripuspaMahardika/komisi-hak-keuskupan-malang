# komisi-hak-keuskupan-malang
Struktur Organisasi Komisi HAK Keuskupan Malang terdiri dari Ketua, Wakil Ketua, Sekretaris I dan II, serta Bendahara I dan II. Kepengurusan ini melayani dan mengoordinasikan kegiatan dialog, kerjasama lintas iman, serta aksi kemanusiaan di wilayah Keuskupan Malang demi terwujudnya persaudaraan sejati dan moderasi beragama.
